# 🌙 My Arch Linux Dotfiles (mangowc)

Custom configuration files for `mangowc` (Mango), `Waybar`, `Kitty`, and more.

## ✨ Configuration Highlights
- **Mango**: Custom window rules (Steam, Teams, Prism, etc.) and performance-optimized `env` variables.
- **Waybar**: Modern, floating "pill" style design with Materia Dark colors.
- **Design**: Premium floating aesthetic with rounded corners and semi-transparent backgrounds.

## 🛠️ Installation / Restore
1. Clone this repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/mangowc-dotfiles.git ~/mangowc-dotfiles
   cd ~/mangowc-dotfiles
   ```
2. Make the install script executable:
   ```bash
   chmod +x install.sh
   ```
3. Run the installer:
   ```bash
   ./install.sh
   ```

## 📦 What's Included?
- `.config/mango/`: Main window manager configuration.
- `.config/waybar/`: Status bar configuration and theme.
- `.config/kitty/`: Terminal emulator configuration.
- `pkglist.txt`: List of installed system packages.
- `aurlist.txt`: List of installed AUR packages.

> [!NOTE]
> This configuration is optimized for NVIDIA + Wayland setups (2026 update).
